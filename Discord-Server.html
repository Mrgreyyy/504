Willkommen im Themen bereich Discord Server

Support Ticket System einrichten

So erstellst du dir ganz schnell und einfach ein Support System für deinen Server

1. Lade den Ticket Tool Bot auf deinen Server ein (Link --> https://tickettool.xyz/)

2. Gehe nun auf die Website vom Ticket Toll und melde dich mit Discord an

3. Suche deinen Server aus und Klicke auf manage, klicke danach auf Panel- Configs

4. Klicke nun oben links auf Create Panel geben nun den Namen des Panels ein und klicke anschließend auf Create Panel

5. Klicke als nächstes auf Settings und wähle die Einstellungen

6. Klicke nun auf Select a Channel to send the Panel und wähle den entsprechenden Kanal wo das Panel hingeschickte werden soll

7. Fertig

Reaction Role erstellen

So erstellst du dir ganz schnell und einfach Reaction roles

1. Lade den Discobot auf deinen Server ein (Link --> https://discohook.org)

2. gebe den Befehl d.reactionrole new in den chat ein

3. Reagiere nun mit dem Emoji das du möchtest auf die Nachicht die die Reaction Role erhalten soll 

4.  Pinge nun die Rolle die der Nutzer beim reagieren auf die Nachicht erhalten soll 

5. Fertig
